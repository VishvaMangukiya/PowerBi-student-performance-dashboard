# 📊 Student Performance & Resource Analysis Dashboard

## 🎯 Project Overview
This Power BI dashboard analyzes student performance and identifies gaps in resource usage.

## 📌 Key Features
- KPI Cards (Average, Highest, Lowest Score)
- Performance Analysis (Good vs Low)
- Gender-wise Comparison
- School Type Analysis (Public vs Private)
- Interactive Filters

## 📊 Tools Used
- Power BI
- Data Visualization

## 🚀 Insights
- Identified low-performing students
- Compared performance by gender
- Analyzed resource gaps
